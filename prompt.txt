Create a simple web app with a Tic Tac Toe game with the following requirements:

- The grid lines should be white
- The circles (O) should be red
- The Xs should be blue

Before starting the game:
- Ask both users for their names

At the end of each game:
- Congratulate the winner
- Show a button to restart with the same users
- Show a button to start over with new users

If users continue playing:
- Keep track of their names
- Keep a counter of games each user won, lost, or tied
